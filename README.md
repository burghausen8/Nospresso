# Nospresso
Projeto de app para venda de cápsulas de café, e acessórios para a mesma.
App criado em Swift UIKit, utilizando as seguintes especificações:
- Padrão de estrutura VIPER 
- Comunicação via protocolos 
- R.Swift 
- ViewCode 
- CleanCode
- Comunicação com API
- LocalStorage
- SnapKit
- Testes Unitários 
- SnapshotTesting
